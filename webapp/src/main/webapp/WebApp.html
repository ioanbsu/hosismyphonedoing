How to delete the application? Below instructions about how to delete this application.
  • If you decided to enable AntiTheft feature and set this application [How Is My Phone Doing?] as Active Device Administrator, then before deleting it you should first remove [How Is My Phone Doing?] app from the list of device administrators: Settings-> Security -> Device Administrators -> Uncheck checkbox next to [How Is My Phone Doing?] app
    After that you can remove application regular way you do it on your device
  • If you did not enable AnthiTheft feature you can remove the application the same way as you remove any other application on your device
How to use this application?
  • After launching this application go to ioanbsu.com
  • Log in on web site your google plus account
  • Make sure that you use the same account to log in on ioanbsu.com and on your android device
  • Right after logging in to ioanbsu.com the application automatically will start search for your devices. The location detection is slow process and sometimes may take long time. If you realize that your device(s) can not be found for long period of time, click Update Devices Location button to force location detection to be executed again
  • To send message click on Send message button, fill in all the required fields and click Send message button inside appeared window
  • To see device status information and update device settings, or use AntiTheft feature click My Devices(*) button and select device you want to check in left list panel
Why I can not see any devices on ioanbsu.com? There are several reasons why you may not see your device on ioanbsu.com website. If you don\'t see any devices on ioanbsu.com, please try doing following:
  • Make sure that you android device has access to the internet
  • Make sure that after closing whit window you see green label at the top of the screen saying that Device is Registered in Google cloud
    • If you see Device is not Registered in Google cloud that means there is a problem registering device into Google Cloud Messaging system. To fix this please try to restart the application. If restarting the application does not help, please send the problem description to ioanbsu@gmail.com
  • Check that google account name that you\'ve logged in with to ioanbsu.com (displayed at the top right corner in ioanbsu.com) the same as your google account name that you use ont you android device. To check the google account name that is used on your device, please close this window and read the text in green at the top of you device\'s screen. it will have information about what account is used by [How Is My Phone Doing?] app
  • Try to refresh ioanbsu.com website
  • If nothing described above helped, please contact the application owner with detailed problem description at ioanbsu@gmail.com
Why am I always see this message when I start the application?
  • People tend to think that this application is a virus because after enabling AntiTheft feature they can not delete the application. Well this is not true, the reason why this application can not be removed once the AntiTheft feature is enabled is because of Android OS policies. Once the AntiTheft feature enabled the [How Is My Phone Doing?] app marked as device administering app. According to Android OS policies it is impossible to remove device administering applications from Android OS. So to remove this app it should we first unchecked as device administering app. After that it can be easily removed.
  • Main purpose of this application is to provide control of your phone/tablet from the web site ioanbsu.com The [How is My Phone Doing?] application should be started only once to register itself in Google Cloud Messaging system. And after that there is no sense of starting and stopping this application - it is already registered in cloud and can be controlled from the website ioanbsu.com . The only reason to open this application would be to see instructions that are displayed in this popup.
